# Yummy

Live demo: https://amiraelsa3id.github.io/Yummy/

Short description

Yummy is a responsive web app that lets users browse meals and view recipes using TheMealDB public API. It’s built with plain HTML, CSS, JavaScript, jQuery and Bootstrap — no build step required.

## Demo

- Live: https://amiraelsa3id.github.io/Yummy/
- Open `index.html` locally in your browser to run the app without a server.

## Features

- Search meals by name or first letter.
- Browse by category, area (cuisine), or main ingredients.
- View detailed recipe instructions, ingredients, tags, and links to source and YouTube.
- Responsive layout using Bootstrap.
- Client-side validation for the contact form.

## Quick start

1. Clone the repo

   git clone https://github.com/AmiraElsa3id/Yummy.git

2. Open the project

   cd Yummy

3. Run locally

- Option A (no server): open `index.html` in your browser.
- Option B (simple HTTP server): from the repo root run:

  - Python 3: `python -m http.server 8000` then visit `http://localhost:8000/`
  - Node (http-server): `npx http-server`

Notes: All required JS/CSS libraries are included in the `js/` and `css/` folders, so no installation is required.
